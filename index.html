Tic-Tac-Toe

Classic Tic-Tac-Toe game (desktop and mobile-friendly).

The Tic-Tac-Toe game project is a client-side application that interacts with a third-party API. The server provides authentication and persists the game objects. A user creates a user account, logs in and can then play tic tac toe and view how many games they've played so far. Additionally, the authentication piece allows the user to change their password and log out. A game is created on the server and 2 players play until one player wins or there are no moves left, ie, a draw.
- Technologies used: JavaScript, jQuery, AJAX, HTML, CSS3, Sass, Bootstrap
